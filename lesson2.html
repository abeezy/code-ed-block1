Adding Background Colors

To add a background color to a page, we add the following code to our body tag: 

<body bgcolor="purple">

We can also use codes to get even more colors! The code for turquoise is #40E0D0. So we just replace the color name with the code: 

<body bgcolor="#40E0D0">

To get more codes, we can use a HTML color picker.

Adding Background Images

To add a background image to a page, we add the following code to our body tag with the location of the image we want to use: 

<body background="http://block1.codeed.org/img/background.jpg">
