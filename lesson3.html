Review

  • Tags: elements
    • opening tag
    • closing tag
  • <html> tells the browser where the HTML starts and stops
  • <head> information about the document that isn't diplayed on the screen
  • <title> name of the document that will show up in the menu bar of the browser
  • <body> everything that is displayed in the browser
  • There are 6 levels of headings:
    • <h1> would be the most important heading
    • <h6> is the least important heading
  • <p> paragraph
  • <br/> line break
  • <hr/> horizontal rule
  • <b> bold
  • <i> curly
  • <u> underlined
  • <body bgcolor="value"> background color
  • <font color="value"> font color

Nesting

Q. What is nesting?
A. placing one element inside another

We want: Coding is for everyone!

This statement is bold and curly.

How do we make this happen? 

<b><i>Coding is for everyone!</i></b>

Images

  • <img> image element/tag
  • Attribute
    • provides additional information about the elements
    • always goes in the opening tag
  • Format: name="value"
    • src="url"
    • height="number"
    • remember to close the image tag with />

Links

A link is text that redirects you to another page when you click on it

  • <a> needs an opening and closing tag
  • Try <a>CodeEd is awesome!>/a>
  • We need the href attribute
    • <a href="url">name of link</a>
    • <a href="http://block1codeed.org/index.html">CodeEd rocks!</a>
  • Remember to always:
    • Use http:// in the URL
    • Put the URL in quotes or the link won't work

    Adding a New Page

    1. Click on the "New" icon or press ctrl + N
    2. This will open an untitled page
    3. Name the file page2.html (or whatever you want)

    What do we need in an HTML document?

    • <html>
    • <head>
    • <title>
    • <body>
    Don't forget the closing tags....

    Q. How would we link our web pages together?
    A. By using <a> ! 

    <a href="page2.html">Page 2</a>
