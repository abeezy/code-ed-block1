How the Internet works

  • The internet connects computers to each other.
  • In order for people to communicate to each other, they use a language called HTML to make webpages.
  • Many people can see a webpage at the same time!

Understanding HTML

What is HTML?

  • HTML allows us to do bold, curvy, big, and colorful text.
  • Without HTML we would just have plain, boring text.
  • You can even do background colors, and use images like this one of a dog:

What does HTML look like?

HTML looks like this:

Only one <b>word</b> in this sentence is bold.
We only <u>under</u>line part of a word here.
<i>This third sentence</i> has an italicized phrase.

And looks like this inside a web browser:


Only one word in this sentence is bold.

We only underline part of a word here.

This third sentence has an italicized phrase.
